
# This library is very simple, since it only has one function called
# ``add_integer()``.


import:
    >>> add_integer = __import__("0-add_integer").add_integer
    >>> add_integer(1, 2)
    3

Negative numbers:
    >>> add_integer(100, -2)
    98

default numbers:
    >>> add_integer(2)
    100

It also works with floating-point values:
    >>> add_integer(100.3, -2)
    98


exit of value:
    >>> type(add_integer(100.3, -2))
    <type 'int'>
